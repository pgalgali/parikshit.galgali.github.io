# Parikshit Galgali — Personal Branding Website

Terminal-aesthetic personal branding site for **Parikshit Galgali**, Senior Associate Director · Enterprise Architecture & Digital Banking Technology at HSBC.

## 🚀 Deploy to GitHub Pages (5 minutes)

### Option A — GitHub Web UI (easiest)

1. Go to [github.com](https://github.com) and create a **new repository** named:
   ```
   pgalgali.github.io
   ```
   *(Replace `pgalgali` with your actual GitHub username)*

2. Click **"uploading an existing file"**
3. Drag and drop `index.html` into the upload area
4. Click **Commit changes**
5. Go to **Settings → Pages → Source → Deploy from branch → main / root**
6. Your site will be live at: `https://pgalgali.github.io`

### Option B — Git CLI

```bash
git init
git add .
git commit -m "Initial deploy: Parikshit Galgali personal branding site"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_USERNAME.github.io.git
git push -u origin main
```

Then enable GitHub Pages in repo Settings → Pages.

## 🌐 Custom Domain (optional)

To use `www.parikshitgalgali.com` instead of GitHub's URL:
1. Create a file named `CNAME` containing just your domain
2. Point your domain's DNS to GitHub Pages IPs
